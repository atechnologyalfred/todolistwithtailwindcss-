# Project Name

A simple todolist project built with TailwindCSS

## Live Demo
https://atechnologyalfred.github.io/todolistwithtailwindcss-/

## Installation
Clone the repository: 

git clone   https://atechnologyalfred.github.io/todolistwithtailwindcss-/

## Setup

Go into the project folder: practice-javascript

cd todolistwithtailwindcss

install dependencies

npm install

## Run the project

npm run watch

## Built With

- HTML
-TailwindCSS
