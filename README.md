# Project Name

A simple todolist project built with TailwindCSS

## Live Demo
https://github.com/atechnologyalfred/todolistwithtailwindcss-.git

## Installation
Clone the repository: 

git clone https://github.com/atechnologyalfred/todolistwithtailwindcss-.git

## Setup

Go into the project folder:

cd todolistwithtailwindcss

install dependencies

npm install

## Run the project

npm run watch

## Built With

- HTML
-TailwindCSS
-Nodejs
